Responsive Grid Systems

What is a grid system?

CSS for columns with margins, which is used to align content (Example »)

Grid systems use relative units (e.g. % and vw) instead of fixed units (e.g. px) so that the layout adjusts according to the viewport width

At narrower breakpoints (e.g. tablet or mobile), each column expands to fill the viewport in a single column

Why do we ♥ grid systems?

  • Easier to Maintain: It's easier to design and code new layouts when the structural guidelines have already been decided
  • Usability: Uniformity and consistency are key to creating a website that your users will find easy to navigate, read and understand
  • Faster to Code: At different breakpoints, we can easily collapse everything into a single column
  • Less 'what if's: Aligning content with a grid means there are less layout exceptions
  • Fluid: Fluid layouts are the way of the future ☺

Coding Your Own Grid System

  1. Determine the maximum width from the design. In our case, that's 1170px. This will be used as a max-width on each row.
  2. Calculate the gutter (the margin between columns) width as a percentage of the maximum width. In our case, that's 2.56% (or 30px/1170px).
  3. Use a grid generator to generate the CSS. (But don't worry — we've done this for you!)

Grid System HTML

Use the row class on each row of content:

<section class="row">

</section>

Within the row, use the column class and the width class (e.g. span_4_of_12) on each column of content:

<section class="row">
    <div class="column span_4_of_12">
        I'm the first of three columns!
    </div>
    <div class="column span_4_of_12">
        I'm the second of three columns!
    </div>
    <div class="column span_4_of_12">
        I'm the third of three columns!
    </div>
</section>

Tip: To use multiple classes on a single element, separate each class with a space in your HTML as shown above.

Gradients

You can now make gradients with CSS3 — no image-editing required!

Use background-image with linear-gradient:

selector {
    background-image: linear-gradient(red, white);
}

The first is the top color, while the second is the bottom color

Additional Animation Effects

Animation Recap

  • Add transition to the before state
  • Remember: transition accepts two values,
    the CSS property and the duration of the animation
Before Selector After Selector 
selector {
    width: 100px;
    transition: width 2s;
}
                            
selector:hover {
    width: 500px;
}

See the Pen Transition: After by Liz Shaw (@anythingcodes) on CodePen.

Animating Multiple Properties

Method 1: Individual Properties

To animate multiple properties (typically at different durations), separate each effect with a comma

selector {
    transition: width 2s, background 5s;
}

See the Pen Animating Multiple Properties - Before by Liz Shaw (@anythingcodes) on CodePen.

Animating Multiple Properties

Method 2: All Properties

Use all to seamlessly transition all properties from before to after states at the same duration

selector {
    transition: all 3s;
}

See the Pen Transitions Using 'all' - Before by Liz Shaw (@anythingcodes) on CodePen.


Sounds great, right? However, all has performance drawbacks, so use it sparingly! (Example »)

Property: transition-delay

Use transition-delay to delay when an animation begins

selector {
    transition: width 1s;
    transition-delay: 3s;
}

See the Pen Transition Delays by Liz Shaw (@anythingcodes) on CodePen.

Property:
transition-timing-function

The timing function changes how fast or slow the animation is at parts of the animation

selector {
    transition: width 3.5s;
    transition-timing-function: ease;
    transition-timing-function: linear;
    transition-timing-function: ease-in;
    transition-timing-function: ease-out;
    transition-timing-function: ease-in-out;
    transition-timing-function: cubic-bezier(n,n,n,n);
}

See the Pen Transition Timing Functions by Liz Shaw (@anythingcodes) on CodePen.

Fun fact: The timing function is sometimes called the speed curve

Property:
transition-timing-function

Additional Notes

If you don't specify a timing function, ease is used

With cubic-bezier, you can create your own timing function

Use cubic-bezier.com to make your own timing function, and compare it to the built-in timing functions (such as ease and linear)

Property: transform

Use transform to rotate, scale, skew, change the perspective, and much more

selector {
    transform: rotate(360deg);
    transform: translate3d(0, -3px, 0);
    /* and more! */
}

transform has too many values to list — use CSS3Maker or CSS3, please! to generate them

A transformation is often animated with transition

See the Pen Transition + Transform - Before by Liz Shaw (@anythingcodes) on CodePen.

Property: box-shadow

The box-shadow CSS3 property is often used for 3D effects, especially when an element is hovered over

selector {
    box-shadow: horizontalShadow verticalShadow blurDistance color;
    box-shadow: 10px 10px 5px grey;
}

Don't worry about memorizing this — use a CSS3 Box Shadow Generator instead!
